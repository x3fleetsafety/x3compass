# Partner Onboarding Flow

## Application phase (Day 0)
1. Partner applies at /partners.html
2. Submission saved to `partner_applications` table (status: pending)
3. Email confirmation sent
4. Reviewed by X3 partnerships team within 24 hours

## Interview phase (Day 1–3)
5. 15-minute Zoom screening with X3 partnerships
6. Application moves to status: interview
7. 50-question knowledge test sent via Typeform
8. Mike Perry reviews application + test result

## Approval phase (Day 3–5)
9. Mike Perry signs off (or rejects with feedback)
10. Application moves to status: approved
11. Welcome email sent with onboarding link

## Activation phase (Day 5–10)
12. Stripe checkout for $499/month subscription
13. White-label configuration (brand, domain, colors) via /partner/branding.html
14. Optional: Stripe Connect onboarding for payment hub
15. Optional: custom subdomain DNS setup (CNAME)
16. Welcome to Partner Discord
17. Kickoff call with dedicated success manager

## Sprint phase (Week 2–8)
18. Week 1: Niche selection — which carrier vertical to target
19. Week 2: ICP definition + 50 prospect list built
20. Week 3: Outreach playbook customized
21. Week 4: First 20 cold messages sent + responses tracked
22. Week 5: First discovery call(s) run with X3 success manager observing
23. Week 6: First proposal sent
24. Week 7: First carrier signed (typical timeline)
25. Week 8: Sprint wrap-up + ongoing-success plan

## Steady state (Month 3+)
26. Monthly office hours with Mike Perry (live Zoom)
27. Weekly Partner Discord activity (deal flow, peer review)
28. Quarterly business review with X3 success manager
29. Annual recertification (Mike Perry sign-off)
30. Annual Partner conference (Year 2+)
